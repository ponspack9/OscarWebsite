Since child I'm passionate about electronics, I started my journey at 6 when I got in my hands my very first console, a GameBoy Color, that was amazing. Years later, I decided to go a step further than playing and tried programming, and all started when I coded my very first program to calculate the area of a triangle with C++. From that day to now I started all kind of projects, just for the joy of learning and programming, from a program to rename and order my music folder to using kalilinux to play with wifi.

I decided to begin the computer engineering degree at UPC in Barcelona, but after two years I realized that my real desire was to program videogames and discovered the degree of videogame design and development, in UPC also, which now I'm currently in.

All the experience and knowledge from both degrees made me learn a lot of languages such as C, C++, java, javascript, python, Assembly (MIPS and intel x86), matlab, C#, SQL, HTML, GLSL and recently learning processing and lua.
And also made me to be keen on and learn 3D modeling and animating, photo edition, arduino, 3D printing, drones, pentesting and of course, videogames.
I also used libraries such as selenium, pandas, numpy, opencv, telegram bot API, AIMA, openGL, webGL, freetype, ImGui, openMP... and now focusing in tensor flow and django.

There are a lot of projects not made with the purpose to be publicated and other simply cannot be shown because of legal terms, but here you will find some of them.

NoReload - VR Shooter [Under development]

This is my last and currently working solo project. This game is aimed to be in Virtual Reality, now is only made for pc as I do not own a VR headset yet.

The objective of the game is to improve your aim while shooting a weapon in VR. This will be done not only shooting at targets, but with puzzles and challenges, all this inside a dark and neon, yet to finish, atmosphere, VR friendly and immersive.

This game was made in 6 days not at fully focus as I'm currently studying.

You can play the first 6 levels as a demo of this game following this link: DO NOT SELECT ANY LEVEL FURTHER THAN 6 IN MAIN MENU

White Panther - God Game

Inpired by the famous thieves band "Pink panthers" we tried to make a game of a jewelry shop that is open at day but you have to keep an eye of it at night as the thieves are about to come.

This project was for the subject "Artificial Inteligence" and was necessarily limited due to it was originaly planned for two people but I did all the game by myself. I kept the upgrading of the shop, restock and hire new employees mechanics, cutting the thieves intervention.

I also managed to make over 30 different customers and a delivery guy that appears every day.

For more information please visit this website I made: White Panther

Memory - Arcade [Under development]

A classical game of memory that will be played in a mobile. There are several features already implemented such as encrypted save data, gameplay with different grid sizes, time, score, advertisment and the game loop

At this project I'm in charge of all development (code, animations, level design...) except from art that is made by the excellent artist Cristina Calero

There's still no actual art placed in the game apart from the fruits (and they are not in the final state), all the elements you can see are placeholders.

This project was started with the idea of making a game with no preassure, giving the best of ourselves to it, and we still find spare time to keep developing it.

1010! Tribute - Casual

This is a university project in which we make 1010! playtesting-ready tribute in C & C++. We primarly used the PugiXML and SDL libraries. The aim to this project is to extract data and analyse it to understand how the player behavies when playing.

It has one simple mechanic: make rows and columns in the grid with the given pieces to gain score.

This project was made for the subject "Interaction and Interface Design" with another 3 colleagues.

You can find more information about this project here: 1010!

Apocaphysics - Shoot'em up [Bullet physics]

Apocaphysics is a time trial car game where you are challenged to destroy all the city in under 60 seconds. If the car is steady for 5 seconds, player loses. Car must keep moving all time. If the increasing side bar gets really hot(red) loses, to reset it you must break something.

This project was made for the subject "Physics 2" with another colleague, with the aim to get used to Bullet physics library.

You can find more information about this project here: Apocaphysics

Davos - Game Engine

Davos is a 3D game engine using OpenGL as well as many libraries such as ImGui.

This project was made for the subject "Game Engines" with another colleague. Our engine have the basic features of an engine, that could be placing models with their texture, transforming them, have a play state, have a file explorer where you can directly import objects, User Interface, multiple cameras, console, inspector and much more such as Quad/Octree culling.

You can find more information about this project here: Davos
